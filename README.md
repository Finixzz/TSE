# TSE
## What is Template Shared Extension ?

Template Shared Extension is out of a box and ready to use service extension that makes .NET Core application development even more rapid. Configurable as is, ready to use, whole application service, and middleware are ready for services to be injected and utilized. Keep in mind that more improvements are incoming and that this is just the initial TSE state!
